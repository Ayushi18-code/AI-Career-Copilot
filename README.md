# AI Career Copilot

## Problem Statement

Students often struggle to evaluate their resumes, identify missing skills, and prepare effectively for interviews.

## Solution

AI Career Copilot is an AI-powered platform that helps users analyze resumes, identify skill gaps, generate interview questions, and receive personalized career guidance.

## Features

- Resume Upload
- Resume Analysis
- Skill Extraction
- Skill Gap Detection
- Interview Question Generator
- Career Recommendations
- ATS Score Estimation
